# Blog-website

A Simple blog website made using  :

    a)  HTML
    b)  CSS
    c)  BootStrap(ver-4.x)
    d)  EJS
    e)  Express.js

Technical Aspects:

    a)  Creating NEW POSTS using compose page.
    b)  Posting new posts on new page and creating new url for new posts
    c)  Rendering new posts on Home page.
